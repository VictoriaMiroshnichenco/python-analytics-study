This study project demonstrates the process of data cleaning, preprocessing, and exploratory data analysis (EDA) using Python.
The goal of the project is to transform raw data into a structured dataset and identify meaningful patterns, trends, and statistical insights.

The project focuses on:

data preprocessing

exploratory data analysis

calculation of statistical metrics

identification of correlations and anomalies

Technologies Used

Python

Pandas

NumPy

Jupyter Notebook

Matplotlib / Seaborn

csv (data source)
